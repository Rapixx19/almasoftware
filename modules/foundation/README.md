# Foundation Module

## Purpose
Core utilities, types, and design tokens that all other modules depend on.

## Exports
- `types/alma.ts` — Domain types (UserProfile, AlmaMessage, etc.)
- `lib/utils/cn.ts` — Tailwind class merging utility
- `styles/alma-tokens.css` — Design tokens (colors, spacing, typography)

## Dependencies
None — this is the base module.

## Used By
All other modules.

## Zone
GREEN — Cursor edits freely.
